=== WP Disable Pages ===
Tags: page
Requires at least: 3.0
Tested up to: 3.5
License: GPLv3

This plugin disables the built-in WordPress Post Type `page`

== Description ==

Apparently not everyone will utilize the built-in WordPress Post Type `page`. This plugin disables it completely.

== Installation ==

Upload the WP Disable Pages plugin to your blog, Activate it.
